# 拖放功能使用指南

## 改进内容

### 1. 修复了鼠标与拖拽组件的位移问题
- 缩小了 DragOverlay 的尺寸
- 优化了拖拽时的视觉反馈

### 2. 改进了嵌套和同级放置的体验

#### 如何同级放置 Layout 组件：
- 拖动 Container 或 Form 到另一个容器的**边缘区域**（外围 8px）
- 会显示**蓝色**高亮，表示将作为兄弟节点放置

#### 如何嵌套 Layout 组件：
- 拖动任何组件到容器的**内部区域**（中心区域）
- 会显示**绿色**高亮，表示将嵌套在容器内部

#### 如何从嵌套中拖出：
- 当容器为空时：拖到边缘 2px 即可拖出
- 当容器有内容时：拖到边缘 8px 即可拖出（更容易）
- 内部 droppable 区域已大幅缩小，方便拖出操作

### 3. 智能碰撞检测
- 使用自定义碰撞检测算法
- 优先检测鼠标指针位置
- 当指针在内部区域时，优先触发嵌套
- 当指针在边缘时，触发同级放置

## 使用技巧

1. **同级放置多个 Container**：拖到容器边缘，看到蓝色高亮即可
2. **Container 嵌套 Container**：拖到容器中心，看到绿色高亮即可
3. **从嵌套拖出**：拖到容器边缘（8px 范围），离开绿色区域即可
4. **普通组件**：拖到容器内部会自动嵌套，拖到边缘会同级放置

## 视觉提示

- 🔵 **蓝色边框** = 同级放置（兄弟节点）
- 🟢 **绿色内框** = 嵌套放置（子节点）
- 📦 **空容器** = 显示"拖放组件到这里"提示
